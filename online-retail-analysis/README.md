# Online Retail Sales Analysis

## Project Overview
Analyzing online retail transaction data to identify bestselling products and peak shopping times to inform inventory and marketing decisions.

**Dataset Source:** [UCI Online Retail Dataset](https://lnkd.in/digH8CnD)

**Tools Used:** Excel + SQL 

**Business Questions:**
- Which products are our bestsellers?
- When do customers buy most?
