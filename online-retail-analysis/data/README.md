# Data

## Raw Data
The raw dataset is not included in this repository due to file size constraints (45MB).

**To reproduce this analysis:**
1. Download the dataset from [UCI Online Retail Dataset](https://archive.ics.uci.edu/dataset/352/online+retail)
2. Place the downloaded CSV file in `data/raw/` folder
3. Rename it to `online_retail.csv` (or note the actual filename)

**Dataset Details:**
- **Source:** UCI Machine Learning Repository / Kaggle
- **Size:** ~541,000 rows, 45MB
- **Format:** CSV
- **Filename:** Online Retail.xlsx or online_retail.csv
